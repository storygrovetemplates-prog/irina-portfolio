# Portfolio Build Brief — Irina Gheorghe

## Stack
Plain HTML, CSS, JavaScript. No frameworks. No build tools.
Host on Netlify. Domain: irinagheorghe.dk

## Fonts
- Cabin Sketch — titles and highlights (Google Fonts)
- Neucha — body and post-it text (Google Fonts)
- Inter — navigation and UI chrome (Google Fonts)

## Texture
Dot grid background: radial-gradient dot pattern, 24px grid, #C8C8C8 dots

## Pages
1. index.html — homepage (FigJam canvas aesthetic)
2. about.html — SWOT canvas
3. Contact lives in footer of homepage

## Interactions
- Custom cursor: whiteboard marker tip SVG
- Post-it stack: click to cycle stats with fly-off animation
- Tool folder: hover to reveal icons with spill animation
- Conference badge: pendulum swing, clicks to LinkedIn
- Hobby cards: draggable with physics drop shadow
- Case study tabs: manila folder tab aesthetic
- Sound design: subtle, medium volume — post-it peel, paper rustle, drag and drop sounds
- Scroll reveal: fade up on enter viewport
- Photo collage: hover tilt on each photo frame

## Typography hierarchy
- Cabin Sketch: hero titles, section headings, sticky note headers
- Neucha: body text, post-it content, annotations, manifesto
- Inter: navigation, buttons, labels, UI chrome

## Case studies
Three projects, no screens shown due to NDA.
Each uses product canvas structure:
problem, users, goals, constraints, decisions, deliverables, outcomes, reflection.
NDA note displayed on each panel.

## Mood
Messy UX process aesthetic. Pencil marks, post-its, tape, paper texture.
Feels like a real designer's canvas, not a clean portfolio template.
Subtle but surprising interactions. Immersive without being distracting.
